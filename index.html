投流大模型 · 全流程总图

总图 · 以账号时间为轴

🗺️ 投流大模型

核心问题：一个号从0到稳定，每一步先干嘛、为什么？
  • 【前期 · 0→标签成型】目标：喂订单、打模型、把标签跑准
    • 投放方式 = 控成本（不放量）新号没模型，放量会被泛流量带歪标签
    • 优化目标 = 进成交ROI（非整体支付ROI）过滤秒退单，人群质量更好
    • ROI：保本 = 售价÷((售价−成本)×(1−退货率))
      • 后台填 = 保本÷0.8躲平台只考核80%的赔付坑
      • 前期基础ROI = 保本×1.3~1.5设高不设低，目的是跑准标签不是消耗
    • 智能优惠券：别开探索期开券→跑到50岁/西瓜头条垃圾流量
    • 人群定向卡严
      • 地域只选常驻、关无关省份路过流量带歪标签
      • 性别年龄卡核心(24-30/31-40)、童装定向宝妈
      • 达人对标同行互动达人
      • 只跑店铺新客、排除自己粉丝
    • 素材结构（按产品定主力）
      • 低毛利多品 → 画面为主占80%靠自然流放大
      • 高毛利单品 → 视频为主占70%靠视频筛精准
      • 种草+收割两套都要、别删素材
    • ★核心：追投喂标签
      • 控成本追投 + 目标直播间成交 + 定向达人30对标
      • 选成交 / 定向达人(A4反推A3)
      • 追投 ≤ 整场30%太多压feed、模型乱
      • 素材筛选漏斗：100烧到40看点击率/ROI→淘汰→留的加2条(到50)→达标加3条→主力每半小时加1条连6次
    • 建模门槛：画面累计1万 / 素材5000 / 出150单 / 花2000-2400
    • 心态：跑不动是正常的(前期双高必然慢)，绝不降ROI；快3-5天慢一周
  • 【中期 · 标签成型→放大】目标：模型立起来、拉高投产
    • 标签精准 → 主计划支付ROI拉到7~8顺序铁律：先准后高
    • 素材跑稳 → 慢慢关追投 → 基础计划承接基础ROI≈追投投产×2+1.5
    • 数据稳 → 可切放量投放
    • 进阶·顺烧破层级（撬自然流）
      • 3前提：feed为主 / 同层级高位 / 有爆品承接
      • 第1支 主计划顺烧打节点(开播20min+20%；300/1h分轮+20%)
      • 第2支 追投矫正铺底(3条控成本、卡30对标)
      • 第3支 一键起量短时冲量(吃回流抢肉)
      • 铁律：每轮看满15分钟、没涨即停
  • 【后期 · 稳定期】目标：稳利润、防被洗、持续放大
    • ROI贴着实跑值往下设(实跑8→7.5~7.8)
    • 新规反制：稳定达标会被掺泛流量洗 → 动态调高投产承接力够硬时
    • 智能券可开(有种草人群后)
  • 【老号变体】区别于新号
    • 模型成熟 → 定向可放宽、可放量、可品牌加热/云图人群包扩量
    • 消耗下滑 = A3高意向薅完 → 用追投拓新客
  • 【诊断速查】
    • 主计划不消耗 = 追投抢量放量追投 / 成交ROI / 只卡性别年龄 三错之一
    • 跑不动 = 样本不够 or ROI太高别降ROI，再喂数据
  • 【贯穿全程 · 红线】
    • 不破保本线
    • 不为消耗降ROI标签带低不可逆
    • 追投跟主计划错开(方式/目标/定向不同)否则抢量
    • 自然流(feed)是主体，追投/素材别压feed

〇、因果链 · 一切从 2 个根推导（卡片化）

地基 · 2个根

一切的源头

根① 标签：谁下单 → 打标签 → 决定流量好坏 → 带低不可逆
根② 流量池：按购买意愿分 内容/交易/广告 → 素材决定进哪池
记住：下面每条规则都从这 2 个根长出来。
框架 · 2个目标

所有动作只服务 2 件事

A·质量(人准不准)：追投 / 定向卡严 / 进成交ROI / 券别开
B·结构(有没有量·能不能转化)：素材结构 / 画面or视频 / 顺烧 / 放量
素材结构在这：追投管"准"(A)、素材结构管"有量+能转化"(B)。只有硬广→薅完掉量；只有种草→被同行收割。两手都要。
推导 · 由根①

保护标签链

标签靠成交喂、且带低不可逆
↓ 所以前期最怕引垃圾人下单
设高不设低 · 跑不动也不降ROI
控成本起步 · 进成交ROI · 券别开 · 人群卡严
结论：这一串其实是同一件事——保护标签别被污染。
推导 · 由根①

追投喂标签链

标签要主动喂"精准成交人"
↓
追投：选成交 / 定向达人(A4反推A3)
追投必须跟主计划"错开"(方式/目标/定向全不同)
否则抢同一池 → 主计划不消耗
结论：追投是"点火喂标签"，错开才不内耗。
推导 · 由根②

素材分工链

素材决定你进哪个池
↓
种草→内容池拉新 · 硬广→广告池收割
所以两套素材都要
别看投产低就删(删冲锋→收割断流→计划崩)
结论：自己种的人自己收，不浪费一分钱。
推导 · 成交密度

顺烧破层级链

短时成交密度高于同赛道
↓
系统把大盘同时段曝光倾斜给你
三支计划顺烧 → 破层级、撬自然流
逆推：3前提(feed为主/同层级高位/有爆品)正是从"顺烧=刺激画面曝光抢大盘"反推出来的。
推导 · 新规

新规反制链

你稳定达标
↓ 平台判定你精准人群吃透了
掺泛流量做"人群扩容"洗你
动态调高投产反制 → 逼系统只给核心用户
结论：平台借你洗流量，你得主动递"我扛不住泛流量"的信号。
推导 · 出价

前期"双高"链

出价决定"排名"，不是"一单扣多少"
↓
前期=高出价抢量打标签 + 高ROI保质量
双高 → 必然慢
结论："跑不动是正常的"——是双高的代价，不是你设错。
推导 · 样本

建模门槛链

标签/模型要够成交样本
画面累计1万 / 素材5000 / 出150单 / 花2000-2400
↓
跑不动多半是"样本不够"，不是"ROI错"
结论：更不能急着降ROI——再喂喂数据。
推导 · 互证

挡羊毛三件套

进成交ROI(过滤秒退)
+ 智能券别开(防任务流量)
+ 人群定向卡严(防杂流量)
结论：三个动作同一个目的——把羊毛/秒退党挡在标签外，互相印证。
推导 · 自然流

自然流优先链

feed(自然流)是主体、付费是辅助
↓ 推出三条都在"别压feed"
追投≤30% · 顺烧要feed为主 · 低毛利主投画面80%
跨卡：画面为主的间(低毛利多品)天生最适合顺烧破层级。
推导 · 新老号

同症不同解链

症状都是"消耗下滑"
新号 = 没模型 → 追投喂标签
老号 = A3高意向薅完 → 追投拓新客
结论：症状一样、根因相反，别用同一招。
推导 · 池流动

池间向上流动链

池子会上下流动
↓ 你持续做对(成交/互动)
种草把内容池浅意向养成成交
"素材两套"=推用户在池间向上流动
结论：种草"养"、硬广"收"，本质是推人从内容池升到广告池。
推导 · 养与收

养与收分工链

追投跑低ROI = "养"量养标签
基础计划设高ROI = "收"割精准
↓
基础ROI 显著高于追投(≈追投投产×2+1.5)
结论：一低一高的分工，对应根②的"种草vs收割"。

隐藏结论 ⟹

能推导出的更深结论

· ROI 不是死值：「保本÷0.8是底线」×「稳定达标会被洗」⟹ 要在保本线与被洗临界之间动态走。
· 「动态调高投产」和「后期贴实跑值往下设」是一体两面：都在追系统当前能给的最优ROI。
· 追投与主计划本就该是两套参数(追投低ROI拓新客 / 主计划高ROI收割)，硬调成一样必然抢量。

一、底层机制（地基）

底层 · 流量池

三大流量池：按购买意愿分

核心问题：用户被分在哪个池？
  • 广告池 = 想买意愿最强 马上下单的高意向人(点小黄车/问价/还搜)
  • 交易池 = 想买意愿中等 有兴趣有可能(点赞评论没下单)
  • 内容池 = 有点兴趣暂不买 浅意向适合种草(看几秒就划走)
  • 系统靠"你的全部行为"实时分池、还会上下流动 看播/赞评/搜索/下单全抓打标签
为什么：全域本质=打通三池、各司其职——种草素材丢内容池拉新(便宜)、硬广丢交易/广告池收割，自己种的人自己收，不浪费一分钱。
底层 · 标签

标签机制：谁下单决定一切

核心问题：系统凭什么给你流量？
  • 链路：谁下单 → 打标签 → 决定后续流量好坏
  • 前期谁下单，决定你后面拿什么流量 宝妈下单→猛推宝妈；乱的人下单→判定成交质量低→越推越差
  • 标签被带低 = 几乎不可逆致命
为什么：为消耗降ROI→引低质量人下单→标签打歪→后期想要好流量"难如登天，救不回来"。所以前期宁可跑不动也不污染标签。
底层 · 新规 ⭐

数据做太好，反而被限流

核心问题：为什么越投越差？
  • ① 稳定达成投产 → ② 判定你精准人群吃透了
  • ③ 启动"人群扩容"掺进泛流量 → ④ 精准度稀释、越投越差
  • 规则：接泛流量→给量；只要精准→压消耗限流
  • 破局：动态调高投产，逼系统只给核心用户解法 投产快摸到稳定值就手动调高
为什么：这不是bug，是平台借千万直播间"洗流量养人群"——商家=免费工具人。你帮平台养人群才给量，不帮就限你。

二、ROI 与 出价

ROI · 设高不设低

ROI 设置全路线

核心问题：后台到底填多少？
  • ① 保本ROI = 售价÷((售价−成本)×(1−退货率)) 例 199/成本60/退货30%→≈2.05
  • ② 后台填 = 真实保本 ÷ 0.890%踩坑 躲平台只考核80%的赔付坑；保本4填5、保本5填6.25
  • ③ 前期 = 保本×1.3~1.5 带自然流保本4→设5~6；投素材保本3→设4.5
  • ④ 中期标签准了 → 拉到 7~8
  • ⑤ 后期贴着实跑值往下设一点 实跑8→设7.5~7.8
  • ⑥ 跑不动也绝不降ROIch7 正解=守保本+追投带消耗
为什么：ROI=向系统点的「流量质量」。设低→系统给垃圾流量、把标签带低不可逆；所以宁可跑不动也设在保本之上。
出价 · 4公式 ★

出价合集：按场景选公式

核心问题：这个出价怎么算？
  • 素材追投·冷启动 → 客单价×毛利率÷0.8 客单100·毛利50%→62.5
  • 追投·有自然流 → 毛利的50%~80% 毛利60→出价30~50
  • 打标签/修正人群 → 客单价的30%~40% 逐条阶梯递增
  • 顺烧/铺底 → 系统推荐值 ±10%
  • 出价决定计划"排名"，非"一单扣多少" 前期必须高、再慢慢降
  • 出价>50 → 预算跟涨≈出价的5倍
为什么：公式不一样不是矛盾、是场景不同；出价高=抢量打标签能力强，所以前期出价必须高，再慢慢降。
新号 · 投流方式

控成本 vs 放量

核心问题：新号先用哪个？
  • ✅ 控成本(新号前期)：卡死成本、质量可控
  • 优化目标选「进成交ROI」 过滤秒退单、人群质量更好
  • ⏳ 放量：起量快但质量不可控、易引泛流量
  • 放量适合"全域数据稳定之后"再放大
为什么：新号先控成本求稳→数据稳了再切放量放大。顺序反了，标签先被泛流量带歪。

三、起新号 与 追投（核心打法）

起号 · 5步+3天

新号起号流程

核心问题：新号怎么开？
  • ① 控成本起步 求稳、把标签跑准
  • ② 优化目标 = 进成交ROI
  • ③ 智能优惠券别开 探索期开券会跑到50岁/西瓜头条垃圾流量
  • ④ 素材结构按产品定主力 低毛利多品→画面；高毛利单品→视频
  • ⑤ 跑不动→用素材追投补种草 两原因：没种草 / ROI太高
  • 打标签3天法：主计划锁死(50/80)、纯靠追投探精准、成交ROI各半1:1ch10
为什么：快3-5天慢一周；起号跑微付费不求快，慢一点反而对，先把标签打扎实后续才能少亏多起量。
追投 · 总纲 ⭐

追投：作用 · 搭建 · 注意

核心问题：追投到底怎么用？
  • 作用：攒精准成交标签、带主计划消耗、矫正人群
  • 选人：①选成交(已成交人群) ②定向达人(蹭对标粉丝) A4反推A3
  • 搭建①：方式=控成本追投(别放量) 放量追投跟主计划没区别→抢量
  • 搭建②：目标=直播间成交(别成交ROI) 探365天首购新客，跟主计划错开
  • 搭建③：定向=选达人、一批30个对标 只卡性别年龄=跟主计划抢量
  • 搭建④：只加"达人"不选"分类"、日预算≈300、跑完换批
  • 🔴 整场消耗追投 ≤ 30% 太多→模型乱、压自然流
  • 建模门槛：追投画面累计成交≈1万；素材累计破5000(素材ROI3→主计划设5) 够样本量模型才立得起来
  • 收尾：素材跑稳→关追投→基础计划承接 基础ROI≈追投投产×2+1.5(追投3→基础7.5)
为什么：追投不是盲目抢量，是喂数据样本；跟主计划"错开"(方式/目标/定向全不同)才不抢量。养出能独立跑的基础计划才是王道。
素材 · 画面vs视频

直播画面 vs 素材

核心问题：主投画面还是视频？
  • 低毛利·多品 → 主投画面、占80%以上 利润低靠自然流放大
  • 高毛利·单品 → 主投视频、占70%以上 靠视频筛精准流量
  • 素材两套都要(种草+收割) 只有硬广薅完掉量；只有种草被同行收割
  • 别看投产低就删素材坑 删了冲锋的→收割的没流量→计划崩
为什么：全域要"种草归种草、收割归收割"；每条素材角色不同，删了冲锋种草的，后方收割的就没人送流量。

四、进阶玩法 与 专项注意

进阶 · 顺烧

顺烧破层级（撬自然流）

核心问题：怎么撬动自然流？
  • 3前提：feed为主 / 同层级高位 / 有爆品承接
  • 第1支·主计划顺烧打节点 开播20min后+20%；300/1h分轮+20%递增
  • 第2支·追投矫正人群铺底 3条控成本、卡30对标
  • 第3支·一键起量短时冲量 刺激消耗和Feed、吃回流抢肉
  • 铁律：每轮观察满15分钟
为什么：用短时高成交密度抢大盘同时段曝光。烧完马上开下一轮会把自然流变付费；某轮没涨立即停。
诊断 · 不消耗

主计划不消耗 = 追投抢量

核心问题：主计划为什么不花钱？
  • ① 追投选了放量追投 跟主计划没区别→抢量
  • ② 优化目标选了成交ROI 跟主计划探同一人群→抢量
  • ③ 定向只卡性别+年龄 覆盖十几亿 vs 21亿没区别→抢量
  • 正解：让追投跟主计划错开(见追投4点)
